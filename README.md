# Novabetx

Novabetx is a modern sports prediction and fantasy-style web application UI created for learning and educational purposes. This project was developed to practice frontend design, backend integration, database handling, deployment, and full-stack web development concepts.

# About The Project

This project focuses on creating a modern mobile-friendly sports platform interface with features such as:

- Responsive UI Design
- Sports Match Cards
- User Authentication System
- Admin Dashboard
- Wallet UI
- Navigation Components
- Database Integration
- Backend Routing
- API Practice
- Deployment Practice

The main purpose of this project is to improve practical coding skills and gain experience in real-world web development workflows.
